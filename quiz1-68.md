## 3. Sorted 2

### รายละเอียดโจทย์

รับ input: 1. `n` = จำนวนตัวเลขทั้งหมด\
2. ตัวเลขใน array\
3. แบ่ง array เป็น 2 ส่วน

-   **Array A** → ตัวที่หาร 5 ลงตัว\
-   **Array B** → ตัวที่หาร 5 ไม่ลงตัว

### เงื่อนไข

1.  หา `max` ของข้อมูลทั้งหมด\

2.  ถ้า `max` หาร 5 ลงตัว

    -   Array A → เรียงจากมากไปน้อย\
    -   Array B → เรียงจากน้อยไปมาก

3.  ถ้า `max` หาร 5 ไม่ลงตัว

    -   Array A → เรียงจากน้อยไปมาก\
    -   Array B → เรียงจากมากไปน้อย

### เงื่อนไขพิเศษ

-   ความเร็วในการ sort มีผลต่อคำตอบ\
-   แนะนำใช้ **Counting Sort**\
-   เวลาในการประมวลผล: `0.00001 วินาที`\
-   Memory Limit: `16 MB`\
-   เขียน Merge Sort แล้วอาจไม่ผ่านเวลา
